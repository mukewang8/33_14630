<h3>课程介绍：</h3>
<p>黑马文学是传智播客开发的专注于电子书阅读的客户端。以不断满足用户燕求、为不同用户提供更好的中文阅读产品，给广大消费者提供更好的阅读体验。</p>
<p>移动web端，有用户模块、书籍模块、后台管理模块，参照了目前流行的jwt认证方式，并结合试下热门的小程序开发．实现了所有接口获取用户信息，部分接口强制登录这样的整套的认证方案。</p>
